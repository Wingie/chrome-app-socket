chrome-socket
==========

Chrome App Sockets (TCP/UDP) made easy! `chrome-socket` simplifies working with the `chrome.socket` for Chrome Apps.

## Installation

`npm install chrome-socket`

## Usage

```js
var socket = require('chrome-socket')
```
