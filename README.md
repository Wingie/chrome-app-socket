chrome-app-socket
==========

Chrome App Sockets (TCP/UDP) made easy! The module simplifies working with the `chrome.socket` API for networking in Chrome Apps.

## Installation

`npm install chrome-app-socket`

## Usage

```js
var socket = require('chrome-app-socket')
```
